### Props

#### AspectRatio

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| ratio | number | 16/9 | Width-to-height ratio |
| class | string | - | Additional CSS classes |

Common ratios: 16/9 (video), 4/3 (standard), 1 (square), 21/9 (ultrawide), 2/3 (portrait)
</Prose>
