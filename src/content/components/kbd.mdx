<div class="space-y-4">
<Prose>
#### Group

Use the `KbdGroup` component to group keyboard keys together.
</Prose>

<ComponentPreview>
  <Fragment slot="code">
```html
<KbdGroup>
  <Kbd>Ctrl</Kbd>
  <Kbd>Shift</Kbd>
  <Kbd>K</Kbd>
</KbdGroup>
```
  </Fragment>
  <KbdGroup>
    <Kbd>Ctrl</Kbd>
    <Kbd>Shift</Kbd>
    <Kbd>K</Kbd>
  </KbdGroup>
</ComponentPreview>
</div>

<div class="space-y-4">
<Prose>
#### Button

Use the `Kbd` component inside a `Button` component to display a keyboard key inside a button.
</Prose>

<ComponentPreview>
  <Fragment slot="code">
```html
<Button variant="outline">
  Accept
  <KbdGroup>
    <Kbd>⌘</Kbd>
    <Kbd>↵</Kbd>
  </KbdGroup>
</Button>
```
  </Fragment>
  <Button variant="outline">
    Accept
    <KbdGroup class="ml-2">
      <Kbd>⌘</Kbd>
      <Kbd>↵</Kbd>
    </KbdGroup>
  </Button>
</ComponentPreview>
</div>

  </InputGroupAddon>

<Prose>
### Anatomy

- `Kbd` - Displays a single keyboard key with proper styling
- `KbdGroup` - Groups multiple Kbd components together
</Prose>
