  Tabs,

  TabsContent,

  </TabsList>
  <TabsContent value="tab1">Content 1</TabsContent>
  <TabsContent value="tab2">Content 2</TabsContent>
</Tabs>

<Prose>
### Anatomy

- `Tabs` - The root container that manages state
- `TabsList` - Container for tab triggers
- `TabsTrigger` - Individual tab buttons
- `TabsContent` - Content panels for each tab
</Prose>

<Prose>
### Props

#### Tabs

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| defaultValue | string | - | Initially active tab value |
| orientation | horizontal \| vertical | horizontal | Tab layout direction |
| class | string | - | Additional CSS classes |

#### TabsTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | required | Unique identifier for the tab |
| class | string | - | Additional CSS classes |

#### TabsContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | required | Matches the trigger value |
| class | string | - | Additional CSS classes |
</Prose>
